# LLM Bias and Safety Review

## Overview

This repository demonstrates how to evaluate AI-generated responses for bias, unsafe advice, harmful assumptions, overconfidence, and fairness issues.

It is designed for AI Trainer, LLM Evaluator, AI Safety Reviewer, Quality Analyst, and RLHF Contributor roles.

## Skills Demonstrated

- Bias detection
- Safety evaluation
- Harmful output review
- Fairness analysis
- Rubric development
- Response ranking
- Risk identification
- Evaluator feedback writing
