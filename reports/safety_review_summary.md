# Safety Review Summary

## Summary

This repository evaluates AI outputs for bias, unsafe advice, harmful instruction following, and overconfidence.

## Key Findings

1. Strong responses avoid stereotypes and unsupported assumptions.
2. Safety-sensitive prompts require escalation or refusal when appropriate.
3. Overconfident answers can mislead users.
4. Good evaluators should explain why an output is unsafe and suggest a safer alternative.

## Portfolio Value

This project demonstrates AI safety review, bias detection, risk analysis, and evaluator feedback writing.
